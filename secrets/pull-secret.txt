This file is a placeholder. The real pull secret lives outside the repo at:

    ~/.local/pullsecret

Kept outside the repo so it never gets accidentally committed or bundled into
a deployable artifact. Scripts that need the pull secret should read it from
$HOME/.local/pullsecret directly.

To populate ~/.local/pullsecret with a fresh secret:

    1. Log in at https://console.redhat.com/openshift/install/pull-secret
       (free Red Hat Developer Subscription — no license needed for dev/test/POC)
    2. Click "Download pull secret"
    3. Move the downloaded file: mv ~/Downloads/pull-secret ~/.local/pullsecret
    4. Verify: python3 -c "import json; print(list(json.load(open('$HOME/.local/pullsecret'))['auths'].keys()))"
       Expected: ['cloud.openshift.com', 'quay.io', 'registry.connect.redhat.com', 'registry.redhat.io']
